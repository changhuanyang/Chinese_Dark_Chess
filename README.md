# Chinese_Dark_Chess
deep learning dark chess agent

# How to Install
```bash
$ cd Chinese_Dark_Chess
$ python3.11 -m venv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
```

# How to play with other human
```bash
$(venv) python run.py
```
